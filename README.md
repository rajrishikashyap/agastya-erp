# Agastya School ERP

A full school ERP system with an AI-powered chatbot, built as a single static HTML file.

## Features
- Dashboard with live fee collection stats
- Student directory with search and filters
- Fee, marks, and attendance records
- SQL-first chatbot with LLM fallback
- Pipeline Inspector showing the RAG architecture

## Stack
- Pure HTML/CSS/JS, no framework
- SQLite-equivalent in-memory data engine
- Groq Llama 3 (pipeline demo)
- 185 students, 6 classes, realistic dataset

## Live demo
[agastya-erp.netlify.app](https://agastyaassignment.netlify.app/)

## Run locally
Just open `index.html` in any browser. No server needed.
